# الگوریتم Gradient Boosting (CatBoost) 

## 1.مقدمه‌ای بر Gradient Boosting
### 1.1 الگوریتم Gradient Boosting چیست؟

 الگوریتم Gradient Boosting یک روش یادگیری ماشین است که برای پیش‌بینی مقادیر (رگرسیون) و تشخیص دسته‌ها (دسته‌بندی) استفاده می‌شود.

ایده اصلی: چند مدل ضعیف (که کمی بهتر از حدس تصادفی هستند) را پشت سر هم می‌سازیم، و هر مدل جدید سعی می‌کند اشتباهات مدل قبلی را اصلاح کند.

مثال ساده: فرض کنید یک گروه دانش‌آموز داریم که باید پاسخ یک سوال ریاضی را بدهند. هر نفر اشتباهات نفر قبلی را نگاه می‌کند و جواب خود را بهتر می‌کند. در نهایت پاسخ نهایی گروه، ترکیبی از تلاش همه است.

عبارات تخصصی:

یادگیرنده ضعیف (Weak learner): مدلی که کمی بهتر از حدس تصادفی عمل می‌کند. مثال: یک درخت کوچک کم‌عمق.

 باقیمانده یا Residual : اختلاف بین مقدار واقعی و مقدار پیش‌بینی شده. هر مدل جدید سعی می‌کند این باقیمانده را پیش‌بینی کند.

  گرادیان یا Gradient : جهت بیشترین افزایش یا کاهش در یک تابع. در اینجا برای کم کردن خطا استفاده می‌کنیم.


## 2. نحوه کار Gradient Boosting (با توضیح قدم‌به‌قدم و محاسباتی)
مرحله ۱: مقداردهی اولیه مدل

ابتدا یک پیش‌بینی ثابت می‌کنیم:

برای رگرسیون: میانگین مقادیر هدف.
مثال: اگر هدف‌ها [3، 5، 7] باشند، پیش‌بینی اولیه = (3+5+7)/3 = 5

برای دسته‌بندی باینری: لگاریتم نسبت احتمال کلاس‌ها (این بخش کمی پیچیده است ولی CatBoost خودش انجام می‌دهد).

مرحله ۲: محاسبه باقیمانده‌ها

باقیمانده = مقدار واقعی - مقدار پیش‌بینی شده

مثال: هدف واقعی = 7 ، پیش‌بینی اولیه = 5 ← باقیمانده = 7 - 5 = 2

مرحله ۳: آموزش یک یادگیرنده ضعیف روی باقیمانده‌ها

یک درخت کوچک می‌سازیم که تلاش می‌کند باقیمانده‌ها را پیش‌بینی کند.

مثال: درخت پیش‌بینی می‌کند باقیمانده = 1.8 برای نمونه قبلی

مرحله ۴: بروزرسانی مدل

پیش‌بینی جدید = پیش‌بینی قبلی + (نرخ یادگیری × پیش‌بینی درخت جدید)

مثال: نرخ یادگیری = 0.1
پیش‌بینی جدید = 5 + (0.1×1.8) = 5+0.18 = 5.18
مرحله ۵: تکرار

این مراحل را چند بار تکرار می‌کنیم تا باقیمانده‌ها کوچک شوند و مدل بهتر شود.

## 3. توابع خطا (Loss Functions) به زبان ساده

برای رگرسیون (پیش‌بینی عددی): میانگین مربعات خطا (MSE)

$$ \[
MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2
\] $$

هر چقدر باقیمانده‌ها بزرگ‌تر باشند، MSE بیشتر است. هدف کاهش آن است.

برای دسته‌بندی باینری: Logistic Loss (خطای لاجستیک)

اندازه‌گیری اختلاف بین احتمال پیش‌بینی شده و کلاس واقعی (مثلاً درست یا غلط).

برای دسته‌بندی چندکلاسه: Softmax Loss

مشابه Logistic Loss اما برای بیش از دو کلاس.

توضیح ساده: تابع خطا به ما می‌گوید مدل چقدر اشتباه می‌کند و Gradient Boosting تلاش می‌کند این خطا را کمینه کند.

## 4. معرفی CatBoost به زبان ساده
### 4.1 الگوریتم  CatBoost چیست؟

الگوریتم CatBoost یک نسخه پیشرفته از Gradient Boosting است که برای ویژگی‌های دسته‌ای (مثل جنسیت، شهر، گروه سنی) طراحی شده است.

مزیت اصلی: دیگر نیازی به تبدیل دستی ویژگی‌های دسته‌ای به عدد (one-hot encoding) نیست.

کارایی بالا و کاهش overfitting دارد.

### 4.2 ویژگی‌های ویژه CatBoost

مدیریت خودکار ویژگی‌های دسته‌ای.

کاهش overfitting با تکنیک Ordered Boosting (ساخت درخت‌ها با ترتیب تصادفی داده‌ها).

آموزش سریع با پشتیبانی GPU و CPU.

عملکرد خوب با پارامترهای پیش‌فرض.

## 5. الگوریتم CatBoost مرحله به مرحله

مرحله ۱: آماده‌سازی داده‌ها

مشخص کردن ویژگی‌های دسته‌ای

نیازی به encode دستی نیست

مرحله ۲: Ordered Boosting

در هر مرحله، درخت جدید با یک ترتیب تصادفی از داده‌ها ساخته می‌شود تا از overfitting جلوگیری شود

مرحله ۳: ساخت درخت‌ها روی باقیمانده‌ها

درخت‌ها به صورت متوالی ساخته می‌شوند

پیش‌بینی‌ها با نرخ یادگیری بروزرسانی می‌شوند

مرحله ۴: منظم‌سازی (Regularization)

تکنیک‌هایی برای جلوگیری از overfitting:

محدود کردن وزن برگ‌ها

افزودن کمی تصادفی‌سازی در ساخت درخت

مرحله ۵: پیش‌بینی نهایی

ترکیب پیش‌بینی‌های همه درخت‌ها برای تولید پاسخ نهایی

ویژگی‌های دسته‌ای با کدگذاری داخلی و بهینه استفاده می‌شوند.

## 6. هایپرپارامترهای مهم CatBoost

| پارامتر         | توضیح ساده                                       |
| --------------- | ------------------------------------------------ |
| `iterations`    | تعداد درخت‌هایی که می‌سازیم                      |
| `learning_rate` | سرعت یادگیری؛ عدد کوچک = آموزش آهسته ولی دقیق‌تر |
| `depth`         | حداکثر عمق هر درخت؛ درخت عمیق = مدل پیچیده‌تر    |
| `l2_leaf_reg`   | جلوگیری از overfitting با جریمه وزن برگ‌ها       |
| `cat_features`  | لیست ویژگی‌های دسته‌ای                           |
| `loss_function` | هدف آموزش مدل (`RMSE`، `Logloss` و غیره)         |

## 7. مزایا و معایب CatBoost
### مزایا

مدیریت خودکار ویژگی‌های دسته‌ای

کاهش overfitting

آموزش سریع

دقت بالا روی داده‌های جدولی

نیاز کم به تنظیم پارامتر

### معایب

آموزش کمی کندتر نسبت به XGBoost برای داده‌های کوچک

مدل کمتر قابل تفسیر است

برای داده‌های خیلی بزرگ بهتر است GPU استفاده شود.

## 8. کاربردهای CatBoost

مالی: رتبه‌بندی اعتبار، تشخیص تقلب

تجارت الکترونیک: سیستم‌های توصیه‌گر

بهداشت و درمان: پیش‌بینی وضعیت بیماران

بازاریابی: پیش‌بینی ریزش مشتری

موتورهای جستجو: رتبه‌بندی صفحات و پیش‌بینی کلیک

## 9. پیاده سازی با پایتون

```
from catboost import CatBoostClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# تقسیم داده‌ها
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# تعریف مدل
model = CatBoostClassifier(
    iterations=500,
    learning_rate=0.05,
    depth=6,
    cat_features=cat_features,
    verbose=100
)

# آموزش مدل
model.fit(X_train, y_train)

# پیش‌بینی
y_pred = model.predict(X_test)
accuracy = accuracy_score(y_test, y_pred)
print("دقت مدل:", accuracy)
```

این کد یک مدل دسته‌بندی CatBoost می‌سازد و آموزش می‌دهد. ابتدا داده‌ها به مجموعه آموزش و تست تقسیم می‌شوند، سپس مدل با مشخص کردن تعداد درخت‌ها، عمق و ویژگی‌های دسته‌ای آموزش داده می‌شود. بعد مدل روی داده تست پیش‌بینی انجام می‌دهد و دقت پیش‌بینی را محاسبه و چاپ می‌کند.

### توضیح پارامترها

iterations=500

تعداد درخت‌هایی که مدل می‌سازد. هر درخت جدید سعی می‌کند خطاهای درخت‌های قبلی را اصلاح کند.

عدد بیشتر ← مدل پیچیده‌تر و دقیق‌تر (ولی زمان آموزش طولانی‌تر).


learning_rate=0.05

نرخ یادگیری؛ تعیین می‌کند هر درخت چقدر روی پیش‌بینی نهایی اثر بگذارد.

عدد کوچک ← آموزش آهسته ولی دقت بالاتر. عدد بزرگ ← سریع ولی ممکن است مدل به overfitting برود.


depth=6

حداکثر عمق هر درخت. درخت عمیق‌تر ← مدل پیچیده‌تر و توانایی یادگیری روابط پیچیده بیشتر.

عدد بزرگ ← ریسک overfitting و عدد کوچک ← ممکن است مدل ساده باشد.


cat_features=cat_features

مشخص می‌کند کدام ستون‌ها ویژگی دسته‌ای هستند (مثل جنسیت، شهر، گروه سنی).

الگوریتم CatBoost این ستون‌ها را خودکار پردازش می‌کند و نیازی به تبدیل دستی نیست.


verbose=100

میزان گزارش پیشرفت آموزش.

عدد 100 یعنی هر ۱۰۰ تکرار، پیام پیشرفت چاپ شود تا کاربر روند آموزش را ببیند

الگوریتم CatBoost: نسخه پیشرفته برای داده‌های دسته‌ای و جدولی

مزایا: مدیریت دسته‌ای، دقت بالا، کاهش overfitting
